Cloudflare Bypass IP Finder (Test Tool)

Description

This tool is a simple Python script created for testing and educational purposes.
It attempts to discover the real IP address of a domain that may be behind services such as Cloudflare.

The user provides a domain and the script performs checks using different methods to attempt to reveal the origin server IP.

Usage

Run the script from the terminal:
python3 ipgrabber.py

Enter the target domain when promoted.
Example: domain:example.com

Requirements:
pip install requests pynput opencv-python pyfiglet
